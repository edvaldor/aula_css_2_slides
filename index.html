🎯 Objetivos da Aula

  • Compreender o papel do CSS na estrutura de páginas web.
  • Aplicar boas práticas de organização de arquivos e blocos CSS.
  • Praticar com exemplos reais.

🧱 Organização Física

/css ├── reset.css ├── base.css ├── layout.css ├── components.css └── responsive.css

Organize o CSS por função. Isso facilita a manutenção.

🎨 Organização Lógica no Arquivo

  1. Reset
  2. Variáveis
  3. Globais
  4. Layout
  5. Componentes
  6. Utilitários
  7. Responsivo

💡 Exemplo HTML

<header class="header"> <h1 class="title">Meu Site</h1> </header> <main class="container"> <section class="card"> <h2>Bem-vindo</h2> <p>Exemplo de CSS organizado.</p> <button class="button">Clique</button> </section> </main>

🧪 Exemplo CSS

/* Reset */ * { margin: 0; padding: 0; box-sizing: border-box; } /* Variáveis */ :root { --primary: #0057ff; --radius: 6px; } /* Globais */ body { font-family: Arial; background: #f0f0f0; } /* Layout */ .container { max-width: 800px; margin: auto; } /* Componentes */ .button { background: var(--primary); color: #fff; border-radius: var(--radius); } /* Responsivo */ @media (max-width: 600px) { .container { padding: 12px; } }

📝 Exercício 1

Reescreva este código com boa organização:

.container {margin: auto; max-width: 1200px;} .button {border-radius: 6px; background: blue; color: white;} body {font-family: Arial;}

🧠 Exercício 2

Crie uma página com dois cards lado a lado. Os cards devem ficar empilhados no celular. Use CSS responsivo e estrutura lógica.
